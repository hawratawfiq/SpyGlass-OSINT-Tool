SpyGlass is an open-source Python OSINT tool developed for cybersecurity professionals and ethical hackers. It automates reconnaissance by collecting publicly available information about a target domain.

## Features

* Email harvesting
* Subdomain enumeration
* Port scanning
* GeoIP lookup
* Sensitive directory detection
* Technology stack fingerprinting
* Packet sniffing (Scapy)
* Keyword leakage detection
* Export results to TXT, HTML, and CSV

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Scapy
* BuiltWith API
* Socket Programming

## Installation

```bash
git clone https://github.com/hawratawfiq/SpyGlass-OSINT-Tool.git
cd SpyGlass-OSINT-Tool
pip install -r requirements.txt
```

## Usage

```bash
python spyglass.py
```

Enter the target domain when prompted.
